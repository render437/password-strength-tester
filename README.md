# Password Strength Tester

A simple web application to test and rate the strength of your passwords. It provides an estimated time to crack and suggestions for improvement.

## Features

*   Enter a password and get an immediate strength assessment (Weak, Good/Moderate, Strong).
*   Estimates the time it would take to crack the password.
*   Provides AI-driven analysis and recommendations to enhance password security.
*   Responsive design for use on various devices.

## How it Works

The application uses JavaScript to analyze the entered password based on length, character types (uppercase, lowercase, numbers, symbols), and common patterns. The estimated cracking time is a simplified representation.

## Technologies Used

*   HTML
*   CSS
*   JavaScript

## Setup and Usage

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/render437/password-strength-tester.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd password-strength-tester
    ```
3.  **Open `index.html`** in your web browser.

Alternatively, you can view a live demo [here](https://render437.github.io/password-strength-tester/) (if you deploy it to GitHub Pages).
